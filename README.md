# 🍵 茶叶冲泡定时提醒程序

一个功能丰富的茶叶冲泡定时提醒应用程序，帮助您掌握完美的泡茶时间，记录品茶体验，并提供个性化的UI定制功能。

## ✨ 主要功能

### 🕐 智能定时提醒
- **多种茶类支持**：绿茶、红茶、乌龙茶、白茶、黑茶、花茶等
- **精准时间控制**：根据不同茶类自动设置最佳冲泡时间
- **声音提醒**：冲泡完成时播放提示音
- **可视化倒计时**：实时显示剩余时间

### 📊 品茶记录与分析
- **详细记录**：记录每次品茶的时间、茶类、口感评分
- **数据统计**：查看品茶频率和偏好分析
- **趋势图表**：可视化展示品茶习惯变化
- **历史回顾**：浏览和管理历史品茶记录

### 🎨 个性化UI定制
- **自定义背景**：选择系统中的任意图片作为应用背景
- **按钮背景定制**：为界面按钮设置个性化背景图片
- **实时预览**：修改设置时即时查看效果
- **一键重置**：随时恢复默认界面风格

### ⚙️ 智能设置管理
- **自动保存**：所有设置和记录自动保存到本地
- **数据持久化**：程序重启后保持所有个人设置
- **配置备份**：设置文件可轻松备份和迁移

## 🚀 快速开始

### 系统要求
- Windows 7/8/10/11
- Python 3.7 或更高版本
- 至少 50MB 可用磁盘空间

### 安装方法

#### 方法一：一键启动（推荐）
1. 双击 `run.bat` 文件
2. 程序会自动检查并安装所需依赖
3. 安装完成后自动启动应用

#### 方法二：手动安装
1. 确保已安装 Python 3.7+
2. 打开命令行，进入程序目录
3. 运行：`pip install -r requirements.txt`
4. 运行：`python main.py`

### 首次使用
1. 启动程序后，选择您喜欢的茶类
2. 点击"开始冲泡"按钮开始计时
3. 在设置页面可以自定义界面外观
4. 品茶后记得在记录页面添加您的品茶体验

## 📖 使用指南

### 冲泡定时器
1. **选择茶类**：在主界面选择要冲泡的茶叶类型
2. **调整时间**：可以手动调整冲泡时间（可选）
3. **开始计时**：点击"开始冲泡"按钮
4. **等待提醒**：程序会在冲泡完成时发出提示音

### 品茶记录
1. **添加记录**：在记录页面点击"添加记录"
2. **填写信息**：选择茶类、评分、添加备注
3. **查看统计**：在统计页面查看品茶数据分析
4. **管理记录**：可以编辑或删除历史记录

### UI定制
1. **背景设置**：在设置页面选择"自定义背景"
2. **选择图片**：浏览并选择您喜欢的图片文件
3. **预览效果**：实时查看背景效果
4. **按钮定制**：同样可以为按钮设置个性化背景

## 🔧 技术特性

- **跨平台兼容**：基于Python和Tkinter，支持多平台运行
- **数据安全**：本地存储，保护用户隐私
- **响应式设计**：界面自适应不同屏幕尺寸
- **模块化架构**：代码结构清晰，易于维护和扩展
- **智能图像处理**：自动调整背景图片尺寸和透明度

## 🎯 使用技巧

1. **最佳体验**：建议使用1920x1080或更高分辨率显示器
2. **图片选择**：背景图片建议选择色彩柔和的图片以保证文字可读性
3. **数据备份**：定期备份`tea_closet/settings.json`和`record/`文件夹
4. **性能优化**：避免选择过大的图片文件作为背景（建议小于5MB）

## 🆘 常见问题

**Q: 程序无法启动怎么办？**
A: 请确保已安装Python 3.7+，或尝试双击run.bat自动安装依赖。

**Q: 自定义背景不显示？**
A: 请检查图片格式是否支持（JPG、PNG、BMP），图片是否损坏。

**Q: 如何备份我的设置和记录？**
A: 复制整个tea文件夹即可备份所有数据。

**Q: 可以添加新的茶类吗？**
A: 可以编辑`tea_closet/tea_F&M.json`文件添加自定义茶类。
